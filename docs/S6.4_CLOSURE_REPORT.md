### What should happen next

The next milestone is **Stage 7 — First External Pilot**. The objective is not more engineering but proving the product with a real user:

> An external firmware engineer, given only the EmberSim README and a HAL header, can go from `git clone` to green CI regression detection in under 15 minutes.

Recommended starting point:

1. Identify one real STM32 project (not a fixture)
2. Walk through: `clone` → `embersim init` → replace HAL boundary → `embersim run` → `embersim baseline create` → `embersim ci-init` → push → green CI
3. Document every friction point
4. Fix the top 1–2 adoption blockers before scaling to more pilots

The S6.4 evidence package supports this: a clean standalone repository, a working CI workflow, and proven regression detection on GitHub Actions.
